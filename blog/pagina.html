Proyectos

Carrito de Compras

Proyecto de Carrito de compras , cada que seleccionamos un producto automaticamente se nos va agregando al carrito donde se mostrara todos los productos con sus precios , cantidad , los precios se sumaran segun valla añadiendose al carrito de igual manera contara con un icono de X para eliminar el producto que ya no desee consumir.

Repositorio GitHub

Diseño de Tarjeta

Proyecto de Diseño de Tarjeta , Nuestra Tarjeta tiene nombre del Producto , una imagen refente al producto que se va a mostrar ,una breve descripcion donde se vera los elementos que contiene el producto ya sea sus elemento o algun mensaje que quiera poner , tendra el precio fijo que cuesta el producto .

Repositorio GitHub

Pagina de Inicio Real Madrid

Proyecto de Pagina de Inicio de Real Madrid , Tendra un encabezado con secciones y dentro de las secciones tendra submenús , una imagen del escudo del real madrid y de fondo una imagenen del equipo de Real Madrid con la champions y en el centro un mensaje de Bienvenida

Repositorio GitHub

Formulario de Registro

Proyecto Formulario de Registro con bordes animados de colores para registrar datos ,con 3 campos de los cuales son usuario , correo electronico y password se usa para registrarse para un login cuenta con 2 opciones , una opcion si ya tiene una cuenta al hacer click le redirige a otro formulario de Sesion y otra opcion de si se olvido la contraseña y por ultimo el boton .

Repositorio GitHub

Colegio Santa Rita de Cassia

Proyecto de colegio a Santa Rita de Cassia , cuenta con una encabezado trasparente con munú y submenus ,una imagen de la insignia del colegio , tiene varias secciones acerca del colegio y una seccion aparte de contactarse para que se pueda contactar con el colegio si le surgio una duda .

Repositorio GitHub
